# Company Website

A modern, responsive company website built with Next.js and TypeScript.

## Features

- Responsive and mobile-friendly design
- Home page with hero section
- Services showcase
- Careers page for job opportunities
- Blog section for articles and updates
- About page highlighting company information
- Contact page for customer inquiries
- Reusable React components
- Optimized performance with Next.js App Router

## Tech Stack

- Next.js
- React
- TypeScript
- CSS/Tailwind CSS (if applicable)

## Project Structure

- Home
- Services
- Careers
- Blog
- About
- Contact

The project follows a component-based architecture for maintainability and scalability.
